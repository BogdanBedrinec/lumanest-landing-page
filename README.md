# LumaNest — Responsive Landing Page

A responsive landing page for a fictional home lighting brand.  
Built with HTML, CSS, and JavaScript, focused on clean UI, structured product storytelling, and conversion-oriented layout.

## Live Demo
https://bogdanbedrinec.github.io/lumanest-landing-page/

## Features
- Responsive landing page layout
- Hero section with strong visual hierarchy
- About section with brand positioning
- Product categories and featured products
- Why choose us section
- Testimonials section
- Email capture / contact CTA
- Clean footer and navigation

## Tech Stack
- HTML
- CSS
- JavaScript

## Project Goal
The goal of this project was to create a realistic portfolio-style landing page similar to a commercial client project, without unnecessary backend complexity.
